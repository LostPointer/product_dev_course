## Frontend (Experiment Portal)

### IMP-F-001 — Добавить колонку «Последняя активность» в список пользователей
**Приоритет:** MEDIUM
**Статус:** [ ] К реализации

Добавить в таблицу админ-раздела (Users) колонку с датой/временем последней активности каждого пользователя.

**Польза:** помогает администратору видеть, кто активно использует платформу, кто давно не заходил.

**Возможная реализация:**
- Добавить поле `last_activity_at` в ответ API `GET /api/v1/users`
- На бэкенде обновлять `last_activity_at` при каждом запросе (или каждые N минут, чтобы не писать в БД при каждом клике)
- На фронте отображать в формате: "2 часа назад" / "вчера в 14:30" / "3 месяца назад"
